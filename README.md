# zeta-mmwave-cli-skeleton

- zeta-mmwave-skeleton을 기반으로 한 프로젝트
- UART 기반 CLI 입·출력 프레임워크 스켈레톤
- TI mmWave SDK의 UART 드라이버 예제 기반
- 레이더 센서 제어 기능은 포함하지 않음
- CLI 입력 수신 및 출력(write) 기능 제공이 목적임

---

## 프로젝트 개요

### 목적
- UART 기반 CLI 인터페이스 제공
- 사용자 입력(command) 수신
- CLI write를 통한 로그 및 응답 출력
- 모듈 제어, 상태 확인, 디버그용 인터페이스 베이스 제공
